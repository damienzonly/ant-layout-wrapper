# Ant Layout Wrapper

This is a template for a sidebar interface.

This is the default structure but you can change it in the `render` method
```
|----------------------------------|
|      |           HEADER          |
|      |---------------------------|
|      |                           |
|      |                           |
|      |                           |
|      |         CONTENT           |
|      |                           |
|      |                           |
|      |---------------------------|
|      |           FOOTER          |
|----------------------------------|
```

Just copy this file in your codebase and use it as root of your entire interface.
